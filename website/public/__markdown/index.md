# O Meu Banco

Finanças divertidas para toda família.

O Meu Banco é um app de educação financeira infantil para pais, responsáveis e crianças. O produto é um simulador educacional e não movimenta dinheiro real.

## Download

- App Store: https://apps.apple.com/app/o-meu-banco-mesada-infantil/id6761734592
- Landing page CTA: https://omeubanco.xyz/#download

## Como funciona

1. Cadastre-se usando Passkey.
2. Adicione seus filhos com nome e avatar.
3. Gerencie mesada, saldo, metas e histórico.

## Principais benefícios

- Educativo: ensina conceitos financeiros de forma prática e lúdica.
- Seguro: não usa dinheiro real.
- Divertido: interface pensada para crianças.
- Familiar: experiência compartilhada entre pais e filhos.

## Segurança

- Privacidade infantil com foco em LGPD e COPPA.
- Sem publicidade na área infantil.
- Login com Passkey.
- Dados protegidos em infraestrutura Google Cloud.
- Área dos pais protegida com PIN e biometria.
